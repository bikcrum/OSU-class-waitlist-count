OSU Class Watch

Live enrollment and waitlist for Oregon State classes · by Bikram

How to use

Add your class CRN(s) in the URL using the crn parameter. Use a comma to separate multiple CRNs.

Format: ?crn=12345,67890

Replace the numbers with your actual CRNs from the OSU class schedule.

Try example: CRN 56130 →

Example

To check one class (e.g. CRN 56130):

{{ request.url_root }}?crn=56130

To check several at once:

{{ request.url_root }}?crn=56130,12345,67890
